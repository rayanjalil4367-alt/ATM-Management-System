
# 🏦 C++ ATM Management System
### Programming Fundamentals Project

A comprehensive, file-based ATM Management System developed in C++. This project simulates a real-world banking experience by separating administrative tasks from customer transactions, all while maintaining data persistence using file handling.

## 🚀 Key Features

### 👤 Customer Portal
* **Secure Login:** PIN-based authentication for individual accounts.
* **Transaction History:** View a detailed log of all past deposits and withdrawals.
* **Financial Services:** Perform withdrawals, deposits, and real-time balance inquiries.
* **PIN Management:** Ability for users to securely change their access PIN.
* **Persistence:** All data is saved to `account.txt` and `transaction.txt`.

### 🛠️ Admin/Manager Portal
* **Account Management:** Create new bank accounts and manage existing ones.
* **System Monitoring:** View all manager activities and global transaction logs.
* **Security & Backups:** Automated data backup system using `account_backup.txt`.
* **Database Control:** View a complete list of all registered bank users and their details.

## 🛠️ How to Run (Visual Studio)
1. Download `atmmanagementsystem.cpp` and all associated `.txt` files.
2. Open **Visual Studio** and create a new **C++ Console App**.
3. Replace the contents of your `source.cpp` with the code from `atmmanagementsystem.cpp`.
4. **Important:** Ensure the following files are in the same folder as your `.cpp` file:
   * `account.txt`, `admin.txt`, `transaction.txt`, `manageractivity.txt`, and `account_backup.txt`.
5. Press **F5** to compile and run.
